函数声明及表达式

· 2019-10-07 · # js

函数声明

function add() {}

函数表达式

const add = function() {}
or
const add = () => {}

区别

foo(); // hello
function foo() {
  console.log('hello');
}

console.log(num); // undefined
var num = 1;
console.log(num); // 1

函数声明存在提升而函数表达式不会。

小结

如果你想在让函数全局通用，在全局作用域上使用函数声明创建函数。
函数表达式可以有效的避免污染全局作用域。比起你的代码中充斥着各种各样的具名函数，匿名函数可以让你使用后立即丢弃掉。使用函数表达式场景主要有IIFE、回调函数。
